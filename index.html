Projects

Fire Detection Vision System

During my placement year, the University of Leeds Real Robotics team was participating in the 2020 MBZIRC challenge, which is a robotics competition held in Abu Dhabi, UAE and I got to be involved. One of the challenges required the use of UAVs to extinguish fires in a high-rise building. I worked in a team to develop a vision system to detect the fires and send their 3D position to the drone, as well as other information about the environment to allow the drone to get into a firing position safely.

Mobile Manipulation Using a Legged Robot

My main placement year project was the development of a mobile manipulation system using a Unitree Laikago quadrupped robot as the mobile platform. This covered a lot of aspects in robotics, manipulator kinematics, computer vision, embedded systems, navigation, 3D printing, and systems integration. Initially, the manipulator's weight was very close to the robot's maximum payload, so I had to design suitable modifications to reduce its weight. The project was cut short due to the COVID-19 pandemic, but various aspects of the system were implemented before that happened.

Robots for disinfection

The COVID-19 pandemic has given rise to an increased need for disinfection, this is a task where robots can excel, since they are not at risk from the virus like humans. In this project the legged robot and a Husky A200 were used for disinfection tasks. I was tasked with setting up the necessary software to control the robot and the onboard pump system and implementing a pose estimation vision system for identification of disinfection targets. The project had news coverage!

Space Impact Recreated in Mbed

A University project that involved developing a game on a NXP KF-64 FRDM board on a handheld PCB gamepad. The developed is a recreation of the Space Impact game that was available on the Nokia 3310 mobile phones. Some additional mechanics inspired from shoot'em up style games were added which include player invincibility, reloading, and unkillable enemies. The game has no concept of an end, instead the player tries to maximise their score during each run until they die. Other basic features such as pause menus, difficulty settings, and a theme were also implemented.

Manipulation Planning In Clutter

University final year project to develop a planner to allow a manipulator to reach and grasp a target object in cluttered environments using manipulation primitives.
